## 视觉特点

### 颜色方案
- 🔵 **蓝色**: 进度、CPU指标、主要操作
- 🟣 **紫色**: 内存指标
- 🟠 **橙色**: 吞吐量、警告
- 🟢 **绿色**: 成功、工作线程
- 🔴 **红色**: 错误、失败

### 动画效果
1. **淡入动画**: 所有卡片从透明到可见
2. **滑入动画**: 从底部滑入视图
3. **脉冲动画**: 实时更新标识
4. **悬停效果**: 卡片放大和阴影增强
5. **进度条动画**: 平滑的进度变化

### 响应式布局
- **手机** (< 768px): 单列布局
- **平板** (768px - 1024px): 2列网格
- **桌面** (> 1024px): 4列网格

---

## 交互说明

### 实时更新
- ⏱️ **更新频率**: 每2秒自动刷新
- 🔵 **更新标识**: 显示"实时更新"或"实时监控"徽章
- 💫 **动画反馈**: 进度条平滑过渡

### 条件显示
- **处理中**: 显示进度监控 + 性能监控
- **已完成**: 显示性能报告
- **失败**: 显示错误信息

### 用户操作
- 📤 **上传文件**: 拖拽或点击选择
- 📥 **下载结果**: 一键下载清洗数据/异常数据/报告
- 🔍 **查看数据**: 切换标签查看不同数据集
- 🔄 **刷新状态**: 自动刷新，无需手动操作

## 数据展示

### 数字格式化
- **千位分隔符**: 10,000 (而不是 10000)
- **百分比**: 45.2% (保留1位小数)
- **内存**: 512MB (整数)
- **时间**: 12.3秒 / 2分30秒 / 1小时15分

### 进度条
- **总体进度**: 大号进度条，显示整体完成度
- **工作线程**: 小号进度条，显示每个线程的进度
- **成功率**: 绿色进度条，显示处理成功率
- **资源使用**: 彩色进度条，显示CPU/内存使用率

### 图标使用
- 📤 上传
- 📊 状态/统计
- 🔄 进度
- 📈 性能
- 💻 CPU
- 💾 内存/数据
- ⚡ 吞吐量
- 👥 线程
- ✅ 成功
- ❌ 错误
- ⚠️ 警告
- 📥 下载
- 🔍 查看

## 用户体验亮点

### 1. 信息层次清晰
- 从概览到详情，逐层展开
- 重要信息突出显示
- 次要信息适当弱化

### 2. 实时反馈
- 处理进度实时更新
- 性能指标实时监控
- 视觉动画提供反馈

### 3. 数据可视化
- 进度条直观显示完成度
- 彩色卡片区分不同指标
- 图标快速识别信息类型

### 4. 响应式设计
- 适配各种屏幕尺寸
- 移动端友好
- 触摸操作优化

### 5. 性能优化
- 条件渲染减少不必要的组件
- 静默失败不影响主要功能
- 缓存机制减少API调用
