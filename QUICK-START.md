# 快速启动指南

## 🚀 5分钟快速开始

### 步骤 1: 安装前端依赖（30秒）

```bash
cd data-cleaning-frontend
pnpm add @radix-ui/react-tabs
```

### 步骤 2: 启动服务（1分钟）

#### 启动后端（如果还没运行）
```bash
cd data-cleaning-service
npm run start:dev
```

#### 启动前端
```bash
cd data-cleaning-frontend
pnpm dev
```

或者使用一键脚本：
```powershell
.\install-and-test-frontend.ps1
```

### 步骤 3: 测试功能（3分钟）

1. **打开浏览器**: http://localhost:5173

2. **上传测试文件**: 
   - 点击"上传文件"
   - 选择 `testdoc/dirty_test_data.csv`
   - 等待处理完成

3. **查看结果**:
   - 进入文件详情页面
   - 滚动到页面底部
   - 看到新的数据查看器！

4. **测试Tab切换**:
   - 点击"清洁数据"Tab → 看到8行清洁数据
   - 点击"异常数据"Tab → 看到7行异常数据
   - 每个Tab都有独立的分页和下载功能

## 🔧 故障排除

### 问题: Tab不显示
```bash
# 解决方案
cd data-cleaning-frontend
pnpm add @radix-ui/react-tabs
```

### 问题: 数据为空
```bash
# 检查后端
curl http://localhost:3100/api/data-cleaning/data/clean/YOUR_JOB_ID

# 检查数据库
cd data-cleaning-service
npx ts-node ../check-database.ts
```

### 问题: 异常数据仍为0
确保后端已应用errorSummary修复：
- 检查 `data-cleaner.service.ts` 是否包含 `errorSummary` 字段
- 重启后端服务
